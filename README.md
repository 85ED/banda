# STATE CYCLE - Site da Banda

Projeto pessoal desenvolvido para a minha banda de rock STATE CYCLE. Site moderno e responsivo com slideshow de fotos, repertório de músicas e informações para contratação de shows.

## Stack Utilizada

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e animações
- **JavaScript Vanilla** - Interatividade
- **Node.js** - Servidor

## Features

- Slideshow automático com 57 fotos da banda
- Transições suaves entre imagens (fade de 2s)
- Word cloud interativa com repertório em 3 categorias
- Design 100% responsivo (mobile, tablet, desktop)
- Menu hamburger para mobile
- Integração com WhatsApp para contratação
- Zero dependências externas
- Performance otimizada (95% de redução no tamanho das imagens)

## Destaques Técnicos

- JavaScript vanilla puro (sem frameworks)
- CSS3 com animações nativas
- Slideshow com algoritmo Fisher-Yates para ordem aleatória
- Otimização de imagens com sips (macOS)
- Código semântico e acessível
